Mockup Generator with GPT

Graphql 구조(Express + Apollo Server + TypeGraphql-Prisma + mariaDB를)이며 csv 와 GTP 를 활용함.

프로젝트를 통해 배운 내용

1. csv 활용
- csv를 작성하기 위해 Excel을 이용하여 정리하고 MySQL Workbench와 HeidiSQL를 이용하여 로컬 및 서버 DB에 데이터를 등록한다.
- 에피소드 :
a. MySQL Workbench의 csv 등록 오류가 있을 때에 인코딩 방법을 변경하여 등록하였음.
b. 로컬 DB에 작업 후 서버 DB에 등록하고자 하였지만 시간이 많이 소요됨에 HeidiSQL를 이용하여 로컬 DB를 서버 DB로 바로 등록하였음.
c. 사용하는 OS 차이로 인하여 DB Table명이 다르게 됨( 첫문자가 윈도우 소문자 / 리눅스 계열 대문자)에 수정하였음.

2. Chat GPT 활용 및 PPT 작성
- Chat GPT를 이용하여 원하는 형태와 내용을 작성하고 PPT로 만들어서 파일을 가질 수 있다.
- 에피소드 :
a. Chat GPT를 이용할 때 원하는 형태와 내용이가 되지 않음에 많은 시행착오가 있었음.
b. PPT 등록을 위해 라이브러리 공부를 하여 사용할 수 있음.

종합 : csv와 Chat GPT를 이용할 수 있으며 다양한 방법으로 활용 할 수 있다.
